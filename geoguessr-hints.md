# GeoGuessr Hint Base

Distilled meta clues per country, for learning and faster plonking. Terse on purpose. **Auto-generated from `data/` — edit the JSON, not this file.**

**Specificity (broad → precise):** `Country` pins the nation · `Region` broad zone (N/S/E/W, biome) · `State` first-level subdivision · `City` specific town · `Special` rare near-pinpoint clue.

**Source codes:** `P` = Plonk It, `GH` = GeoHints, `W` = Wikipedia, `GT` = GeoTips. Each hint is flagged so the webpage can filter by source + specificity.

Countries are grouped by continent, ordered biggest → smallest by area. Per-country **Study** links: Plonk It page (has clickable Street View images) + GeoGuessr map to practice immediately.
